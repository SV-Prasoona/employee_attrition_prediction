# Employee Attrition Prediction

## Overview
Machine Learning project to predict employee attrition using Random Forest.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## Steps
1. Data preprocessing
2. Feature engineering
3. Model training
4. Evaluation
5. Deployment

## Run Project

Train:
python main.py

Run App:
streamlit run app/app.py
